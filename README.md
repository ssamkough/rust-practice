## Getting Started (sorry Windows)

Installing rust:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Compiling Rust file:

```bash
rustc hello-rust.rs
```

Running compiled file:
```bash
./hello-rust
```
